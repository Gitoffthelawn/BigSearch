大术专搜 可以通过以下几种方式使用：

  1. 工具栏按钮弹出界面（推荐）
  2. 选择文本后右键菜单
  3. Firefox侧边栏（施工中）

由于侧边栏可用但未完善，您可以先关闭侧边栏，使用工具栏按钮。

当然，继续在侧边栏中使用也可以～

You can use Big Search via:

  1. Toolbar button popup (recommended)
  2. Context menu after selecting text
  3. Firefox sidebar (in construction)

Sidebar is available and not 100% completed, so you can close sidebar and use toolbar button.
