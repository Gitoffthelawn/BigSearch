大术专搜 可以通过以下几种方式使用：

  1. 工具栏按钮弹出界面（推荐）
  2. 选择文本后右键菜单（菜单行为可设置）
  3. Firefox侧边栏
  4. 单独的标签

一般来说，推荐先关闭侧边栏，使用工具栏按钮。

当然，继续在侧边栏中使用也可以～


对了，界面有10+种颜色主题，可以在下方选择一个顺眼的

You can use Big Search via:

  1. Toolbar button popup (recommended)
  2. Context menu after selecting text (behavior configurable)
  3. Firefox sidebar
  4. Standalone tab

Generally, we recommend to close sidebar and use toolbar button first.

Insist using in sidebar is okay, however ~
