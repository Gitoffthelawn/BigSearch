Enable context menu (need restart/reenable) :
Big Search can show a menu item when you right-click on web.
  • When context menu clicked, set selection text as search term, then: (need restart/reenable)
    • Open popup (Not on Chrome-like browsers. Chrome only allow popup to be opened via user clicking toolbar button only. So, as a workaround, you may want to use key shortcut combinations instead)
    • Open sidebar (Firefox only)
    • Open Big Search standalone tab
When context menu clicked (or keyword shortcut used), put selection text to clipboard also (need restart/reenable)


Show when new version available, at corner of addon page (need restart/reenable)

Disable communication with online editing-engine GUI web (need restart/reenable)
Disabling it will give you more security (if there is any potential security hole), but you'll have to manually copy-paste JSON
